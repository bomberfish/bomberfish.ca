Okay, that was kind of a lie. But let me explain.

I occasionally use tools such as Apple's proofread feature, and other tools such as Kagi Doc (which is in a private preview as of the writing of this post), to help polish an already-written post. I've also used AI to help create some of the features and elements on this webpage. Notably, I used AI to create the animated interactive background, and the Google Photo Sphere viewer used in the <TransitionLink href="/blog/google-glass-retrospective" class="router-link">Google Glass post</TransitionLink> (although, that actually wraps an existing library). It's also really useful for cleaning up code and migrating library versions.

But, I would like to make some commitments here.

First of all, I will never fully AI-generate any text or image on my blog. If I refine a post using AI tools, I will commit both the unedited and edited versions to the git repository. You can see an example of such a diff [here](https://github.com/bomberfish/bomberfish.ca/commit/31f52ec4b4ac9f5940b48da0bc9fae6bba8f0d1d). If the cover images are enhanced with AI-powered tools (such as Topaz Photo for upscaling), I will disclose it as such in the image caption.

This blog is a place for me to have a place to express myself, and I intend on keeping it that way.
